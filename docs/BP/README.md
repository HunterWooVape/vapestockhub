## 1. 文档目的

本目录用于集中沉淀 `VapeStockHub` 商业计划书相关的核心母版内容，覆盖：

- 正式 `BP / PPT` 文字母版
- 路演讲稿
- 面向不同对象的差异化版本

本目录的目标不是只做一份“好看”的 BP，而是建立一套：

- 可复用

- 可针对不同场景重组
- 可持续迭代

的商业表达资产库。

1. 先看 `01_BP_MASTER_DECK.md`
2. 再看 `02_PITCH_SCRIPTS.md`

4. 如果遇到深问，再补充三个附录

## 4. 当前母版的核心统一口径

整套 BP 当前统一采用以下母线：

```text
VapeStockHub is a B2B inventory liquidity network for the global vape supply chain.
```

中文统一口径：

```text
VapeStockHub 是一个面向全球电子烟供应链的 B2B 库存流动性网络。
```

所有对外叙事，原则上都应围绕以下逻辑展开：

1. 行业存在长期、重复发生的库存流动性问题
2. 这个问题过去存在，但现在第一次具备被数字化组织的条件
3. 我们不是在做普通 inventory listing site，而是在建立流动性层
4. MVP 先验证流动性，不先模拟完整 marketplace
5. 商业模式先变现流动性，再变现访问权与基础设施

---

## 5. 写作与演示原则

- 对外公开或路演主文案优先使用英文
- 内部说明、中文讲解、策略判断保留中文
- 不把项目讲成“普通电子烟网站”
- 不把项目讲成“什么都做的大平台”
- 不夸大当下规模，不伪造成熟 traction
- 强调：
  - `inventory liquidity`
  - `structured supply`
  - `qualified demand`
  - `supply-first GTM`
  - `layered monetization`
