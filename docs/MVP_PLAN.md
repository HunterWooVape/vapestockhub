# VapeStockHub MVP 执行计划（Single Source of Truth）

## 1. 目标与边界
- 目标：以最小成本跑通“库存信息展示 → 私域联系 → 有效询盘”闭环。
- 北极星指标：有效询盘数（Weekly）。
- MVP边界：仅做展示与引流，不做注册入驻与在线交易闭环。

## 2. 已确认决策
- 首批上线库存量：60条。
- 首发页面策略：市场页、品牌页、价格带页同步发布。
- 品牌页阈值：库存驱动品牌页“近30天上架SKU≥2”。
- 风险框架：双轴模型（商业机会分 + 监管风险分）。
- 手动SLA：48/72/96三档复核。
- 多语言：MVP阶段仅英文；第二阶段再决策 Spanish/Arabic。
- UI主题：Dark Pro。
- UI强调色：Teal（#22C7A9）。
- UI字体：Inter。

## 3. 工作顺序（6周）
- 第1-2周（运营底座）：国家评分表v1、品牌池清单、库存复核台账、首批60条库存清洗。
- 第3-4周（流量底座）：详情页/市场页/品牌页/价格带页模板上线，索引与noindex规则生效。
- 第5-6周（转化底座）：询盘归因字段、渠道追踪、周度看板、SLA复核流程跑通。

## 4. 上线验收标准
- 功能验收：用户可从任意首发页面进入详情页并完成私域联系动作。
- 数据验收：可记录来源页面、来源地区、来源渠道、首响时间。
- 运营验收：60条库存全部有状态且进入48/72/96复核节奏。

## 5. 周复盘固定看板
- 有效询盘数（周）
- 询盘首响时效（分钟）
- 页面到联系点击率（Page→Contact CTR）
- 询盘转化率（询盘→有效商机）
- 库存有效率（active且在复核时效内）
