# VapeStockHub 议题暂存清单（Parking Lot）

## A. 暂缓到MVP后讨论
- 多语言第二阶段优先顺序（Spanish vs Arabic）。
- 外部卖家投稿开放机制与审核流程。
- 会员费/发布费/佣金的商业化细则。
- 自动化SLA系统（替代手动48/72/96）。
- 更细分国家策略与本地化运营脚本。

## B. 需要数据验证后再定
- 品牌页扩张速度与淘汰机制阈值。
- 市场页与价格带页索引放量节奏。
- 私域分群是否从地区扩展到“地区+品类”双维。

## C. 触发条件（满足再启动）
- 连续4周有效询盘稳定增长。
- 首批市场页/品牌页产生可复现的询盘来源。
- 私域团队响应时效与跟进成功率达到目标线。
